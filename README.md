# Student Hub

Welcome to the centralized resource repository for course materials! This repository contains files and resources that students will need throughout the course, separate from specific GitHub Classroom assignments.

## Repository Structure

### [`data/`](./datas/)

Example datasets for use in assignments, practice exercises, and demonstrations.

- Includes documentation for each dataset
- Various formats (CSV, RData, etc.)
- Ready to use in your analysis scripts

### [`demos/`](./demos/)

In-class demonstration scripts and code examples.

- Commented code from class sessions
- Working examples of key concepts and techniques
- Reference implementations for common tasks

### [`templates/`](./templates/)

Templates for project submissions and course documents.

- Project submission templates
- Report templates
- Must be **copied, completed, and submitted** with assignments

## Getting Started

### Accessing Resources

1. **Clone this repository** to have local access to all resources:
   ```bash
   git clone https://github.com/d2m-r/student-hub.git
   ```
2. **Browse online** to view individual files without downloading
3. **Download specific files** using the GitHub interface

### Using Templates

**Important:** Templates should NOT be edited directly in this repository!

1. Navigate to the [`templates/`](./templates/) folder
2. Copy the template you need to your assignment repository
3. Complete the template with your own information
4. Submit as part of your assignment

### Running Demo Scripts

Demo scripts in the [`demos/`](./demos/) folder are provided as learning resources:

- Read through the comments to understand each step
- Run the scripts to see examples in action
- Modify them for practice and experimentation
- Use them as starting points for your own work

## Contributing

This is a course resource repository. Students should not modify files directly but can:

- Report issues with datasets or scripts
- Suggest additional resources
- Ask questions about usage
