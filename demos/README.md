# Demo Scripts

This directory contains scripts and code examples demonstrated during class sessions.

## Purpose

These demos are designed to:
- Illustrate key concepts covered in lectures
- Provide working examples of techniques and methods
- Serve as reference implementations for assignments

## Organization

Demo scripts are organized by topic or class session. Each demo includes:
- Well-commented code explaining each step
- Example output or expected results
- Any required data files or dependencies

## Usage

Students can:
- Run these scripts to see examples in action
- Use them as templates for their own work
- Refer to them when completing assignments
- Modify them for practice and experimentation

Make sure to have all required packages installed before running demo scripts.
