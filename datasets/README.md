# Datasets

This directory contains example datasets for use in assignments and practice exercises.

## Available Datasets

Each dataset includes:
- The data file(s)
- A README or documentation file explaining the dataset structure
- Information about the source and any relevant context

## Usage

Students can download these datasets directly or reference them in their analysis scripts. These datasets are provided for educational purposes.

## Adding New Datasets

When adding a new dataset, please include:
1. The dataset file(s) (CSV, TSV, RData, etc.)
2. A README or documentation file
3. Information about data source and any licensing
4. Description of variables/columns
